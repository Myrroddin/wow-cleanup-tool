# Changelog

This file summarizes the most recent changes for the current release. For a full history, see [GitHub Releases](https://github.com/Myrroddin/wow-cleanup-tool/releases).

## v1.0.0 (2025-11-18)

### Major Improvements
- All log messages are now fully localized (including verbose output)
- Folder Cleaner tab refreshes immediately after processing folders or screenshots
- Game Optimizer verbose logging shows individual Config.wtf changes
- Executable naming and versioning fully automated in GitHub Actions
- First-run experience now prompts for WoW folder and does not inherit developer settings

### Bug Fixes
- Fixed: Folder Cleaner tab not refreshing after deleting screenshots or folders
- Fixed: Game Optimizer status color now updates with theme after applying preset
- Fixed: No hardcoded WoW paths in builds; settings files are user-specific
- Fixed: All log messages and dialogs are localized

### Documentation
- Updated README.md, Technical-Details.md, and LOCALIZATION.md to reflect new features and fixes
